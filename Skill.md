# Invoice Checker Skill

## Objective
Provide a step-by-step procedure to check invoices for correctness.

## Steps
1. Read the invoice data.
2. Check required fields.
3. Validate calculations.
4. Identify inconsistencies.
5. Generate a clear report.

## Quality Checks
- Ensure totals match line items.
- Flag missing or invalid data.

## Output Format
- Status: PASS or FAIL
- List of detected issues
